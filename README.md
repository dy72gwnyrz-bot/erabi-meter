# どれどれ

**公開URL: https://dy72gwnyrz-bot.github.io/erabi-meter/**(GitHub Pages で公開中)

iPhone の場合は上記URLを **Safari** で開き、「ホーム画面に追加」するのがおすすめです(手順は後述)。

たくさんの候補を、少ない比較(1対1〜4択のデュエル形式)で順位づけする、日本語UIのシンプルなPWAです。
比較のたびに Elo レーティング(1500基準)で全候補のスコアを更新し、最後にランキングとして表示します。

- オフラインでも動作します(Service Worker によるキャッシュ)
- 通信も外部送信もありません。すべてのデータは**この端末の localStorage にのみ**保存されます
- サーバー・アカウント登録は不要です
- 用途別のひな形と30秒のお試しから始められます
- 比較を始める前に「さくっと / ほどよく / じっくり」の3段階から、くらべる量を選べます
- 結果のコピー・共有・画像保存、データの書き出し・読み込みに対応しています

## ファイル構成

```
erabi-meter/
```

## ローカルでの動かし方

普段使うだけなら上記の公開URL(GitHub Pages)を開けば十分です。以下はコードを手元で確認・改造したい場合の手順です。

PWA の機能(manifest / service worker)は `file://` では正しく動作しないため、簡易サーバーで配信してください。

### Python がある場合

```sh
cd erabi-meter
python -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開きます。

### Node.js がある場合

```sh
cd erabi-meter
npx serve .
```

表示された URL(例: `http://localhost:3000/`)を開きます。

## iPhone で「ホーム画面に追加」する手順

1. iPhone の **Safari** で、公開URL(`https://dy72gwnyrz-bot.github.io/erabi-meter/`)を開く
   - ローカルサーバー(例: `http://<PCのIPアドレス>:8000/`)でも開けますが、`http://` かつ非 `localhost` のアドレスは Service Worker が動作せず、オフライン非対応になります。手元で試す場合もこの制約を理解した上で使ってください
2. 画面下部の **共有アイコン**(四角から矢印が上に伸びたマーク)をタップ
3. メニューをスクロールし、**「ホーム画面に追加」** をタップ
4. 表示名（「どれどれ」）を確認し、右上の **「追加」** をタップ
5. ホーム画面にアイコンが追加されます。以後はアイコンから起動すると、アドレスバーのない全画面(standalone)のアプリとして動作し、オフラインでも使えます

> 注: 「ホーム画面に追加」は Safari からのみ可能です(Chrome など他ブラウザでは表示されません)。

## データについて

- 作成したランキング、比較の進行状況、これまでの結果履歴はすべて **ブラウザの localStorage** に保存されます
- サーバーには一切送信されません
- ブラウザのデータ消去・サイトデータ削除・アプリの再インストールなどを行うと、保存内容は失われます
- 別の端末やブラウザとデータは同期されません
